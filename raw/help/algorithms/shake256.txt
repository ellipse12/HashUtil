
---------------SHAKE256---------------

--Secure Hash Algorithm And Keccak 256--

Returns a specifide bit length hash digest from a given string using the SHA-3 algorithm

Notice: SHA3 is the strongest current hashing algorithm and it is recommended for practical use.

Wikipedia Article: https://en.wikipedia.org/wiki/SHA-3

Usage: hash <string> [SHAKE256, 16] <length (default is 32)>

Note: number of output bits = length*8
